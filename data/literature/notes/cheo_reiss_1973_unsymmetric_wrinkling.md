# Cheo & Reiss 1973 — Unsymmetric Wrinkling of Circular Plates

**Источник:** `1_CheoReiss_UnsymmWrinkling_rotated.pdf`

## 1. Что именно важно для проекта
- Это ключевой plate-source для логики **ветвления неосесимметрического состояния от нелинейного осесимметрического фона**.
- Полезен как ближайший источник для классической plate-постановки, с которой ты уже концептуально сравнивал свою систему.
- Особенно важен как исторический и методический ориентир для:
  - нелинейной осесимметрической background-задачи;
  - линеаризации относительно неё;
  - поиска wrinkling loads как eigenvalues;
  - сравнения с Panov–Feodosiev.

## 2. Какие обозначения используются
- Используются `W(r,θ)` для прогиба и `F*(r,θ)` для функции напряжений.
- После безразмеривания появляются `w(z,θ)`, `f(z,θ)` и параметры нагрузки.
- Неосесимметрические возмущения разделяются по `sin(nθ)` / `cos(nθ)`, а радиальные функции обозначаются как `y_n(z)`, `z_n(z)` (или близко к этому в разных формулах статьи).
- Это уже ближе к von Kármán plate language, а не к твоему mixed-weak shell language.

## 3. Какие уравнения / страницы критичны
- **С. 75–76 / начало статьи** — abstract + постановка wrinkling problem и связь с axisymmetric state.
- **Раздел 2** — Problem B и Problem S; именно здесь задаётся нелинейный осесимметрический фон.
- **Раздел 3** — eigenvalue problem после линеаризации около symmetric state.
- **Формулы (3.5)–(3.6)** — главный блок для мод `n`: ODE + boundary conditions.
- **Раздел 5 / Fig. 4 / Table I** — численные wrinkling loads.
- **Раздел 5, обсуждение Panov–Feodosiev** — очень полезно как пример того, как неточный ansatz даёт ложные ветви.

## 4. Где возможны расхождения с твоей моделью
- Это **круглая пластина в теории фон Кармана**, а не общая непологая оболочка вращения.
- Нет mixed-weak переменных и независимых каналов `(v,S)` / `(psi,H,chi)`.
- Граничные условия в статье — для clamped circular plate; у тебя сейчас критично сравнение с `simple support`.
- Поэтому источник лучше использовать для **проверки структуры задачи**, а не как конечную формульную базу.

## 5. Можно ли использовать как основной источник
- **Да, как основной plate-source для wrinkling / bifurcation logic.**
- **Нет, как основной источник финальных shell-equations проекта.**
- Статус: **очень важный вспомогательный**.
