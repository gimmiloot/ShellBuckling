\begin{abstract}
This note presents the current mixed-weak version of the shell-buckling model used in the repository.
Its purpose is discussion and synchronization, not full formal derivation.
The emphasis is on the current unknowns, the active equation structure, the boundary conditions, the present criticality criterion, and the current solution procedure.

\section{Current criticality criterion}
The current criterion is based on the mixed-weak boundary matrix
\[
B_{\mathrm{mix}}(q),
\]
constructed from two central regular modes of the active linearized problem.

The current working criterion is
\[
\sigma_{\min}(B_{\mathrm{mix}}(q)) = 0.
\]

\paragraph{Important status note.}
At the current stage this should be treated as the active project criterion of the mixed-weak branch.
It should not automatically be read as a fully finalized physical criterion for the complete shell problem outside the current verification boundary.

\section{Solution procedure}
A compact description of the present workflow is:

\begin{enumerate}[leftmargin=2em]
    \item Compute the axisymmetric prebuckling background.
    \item Build the current mixed-weak linearized operator around that background.
    \item Construct the two central regular modes.
    \item Form the mixed-weak boundary matrix \(B_{\mathrm{mix}}\).
    \item Scan the load parameter.
    \item Detect the candidate critical state through the degeneration of \(B_{\mathrm{mix}}\), currently monitored through \(\sigma_{\min}(B_{\mathrm{mix}})\).
\end{enumerate}
