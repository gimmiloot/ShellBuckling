proof pilots currently integrated in the repository.

\section{Introduction}
The current active direction of the project is the mixed-weak branch.
It replaced the older reduced/full architecture after that route failed to give
a qualitatively new and stable main criterion.
The purpose of the present note is not to re-derive the shell theory from zero,
but to give a compact supervisor-facing summary of the current repository-level
object that is actually being checked and used.

\section{Problem statement}
The repository currently studies the linearized non-axisymmetric problem around
a corrected axisymmetric prebuckling background, within the mixed-weak branch
and its present simple-support-oriented testbench boundary conditions.
This note concerns the current repository boundary only:
\begin{itemize}[leftmargin=2em]
    \item the active mixed-weak operator class,
    \item the current boundary-pair logic,
    \item the present center-regular mode construction,
    \item the current mixed boundary matrix $B_{\mathrm{mix}}$,
    \item the current working spectral indicator $\sigma_{\min}(B_{\mathrm{mix}})$.
\end{itemize}
It is not a final theorem for the full closed shell BVP and it does not certify
a final physical simple-support critical load.

\section{Unknowns and Mixed-Weak Structure}
\paragraph{Accepted working structure.}
The retained working base of the branch is the corrected kinematics together
with the corrected circumferential bending block already fixed in the main
theory notebook.
The active mixed-weak unknown/resultant blocks are
\[
U = (u_s, u_n, v, \varphi, \psi),
\qquad
P = (T_s, T_\theta, S, Q_s, M_s, M_\theta, H, \chi).
\]
In the active solver cores the primary coefficient vector is realized through
\[
(u_s, u_n, v, \varphi, \psi, T_s, Q_s, M_s),
\]
while $T_\theta$, $S$, $M_\theta$, $H$, and $\chi$ are reconstructed
algebraically from those primary fields and the axisymmetric background.

A central feature of the branch is the presence of two independent
circumferential channels,
\[
(v,S), \qquad (\psi,H,\chi),
\]
which are treated as genuine parts of the operator class rather than as
redundant consequences of an older closure logic.
Pilot 02 closes this separation locally at the level of the current formulas and
abstract witness logic.

The current basic kinematic measures retained in the theory file are
\[
\varepsilon_s(U) = u_s' - \kappa_{s0}u_n,
\qquad
\varepsilon_\theta(U) = \frac{c_0u_s+s_0u_n}{x} + \frac{n}{x}v,
\]
\[
\gamma_{s\theta}(U) = v' - \frac{r_0'}{r_0}v - \frac{n}{x}u_s,
\qquad
g_n(U) = u_n' + \lambda_{s0}\varphi - \kappa_{s0}u_s,
\qquad
\kappa_s(U) = \varphi'.
\]
Here $c_0=\cos\varphi_0$, $s_0=\sin\varphi_0$,
$\lambda_{s0}=1+e_{s0}$, $\lambda_{\theta0}=r_0/x$,
and $\kappa_{s0}=\varphi_0'$ are background quantities.

\section{Current System and Repository-Level Clarified Formulas}
The active mixed-weak operator is read at repository level through
\[
A_n(X, \widehat X; q)
 = K_n(X, \widehat X) - G_{\mathrm{ps},n}(X, \widehat X; q)
   + B_{\mathrm{partial},n}(X, \widehat X) = 0,
\qquad X=(U,P).
\]
This is the current project statement of the operator form.
The full article-level derivation of each term is still open, but the branch is
no longer described as a single scalar prestress potential $G(U)$.

\subsection{Algebraic Mixed-Weak Reconstructions}
The active solver cores implement the following current reconstructions:
\[
r = c_0u_s+s_0u_n,
\qquad
S = \frac{v' - a_0v - (n/x)u_s}{2(1+\nu)},
\]
\[
\kappa_\theta^{\mathrm{new}}
 = \frac{c_0}{r_0}\varphi - \frac{s_0}{r_0^2}r + b_\theta\psi,
\qquad
M_\theta = \nu M_s + \frac{\kappa_\theta^{\mathrm{new}}}{\Lambda},
\]
\[
H = \frac{\psi' - a_0\psi - b_s\varphi}{C_{\mathrm{twist}}},
\qquad
\chi = \frac{b_\theta M_\theta - H' - 2a_0H}{\lambda_{\theta0}}.
\]
These relations are the basis of the current channel split checked in pilot 02.

\subsection{Repository-Level Closed Statement of the Prestress/Load Block}
The strongest current repository-level closed statement of the active
prestress/load block is
\[
G_{\mathrm{ps},n}^{\mathrm{repo}}(X,\widehat X;q)
:= \int_{x_0}^1
\bigl( \widehat T_s\, g_s^{\mathrm{repo}}(X;q)
     + \widehat Q_s\, g_n^{\mathrm{repo}}(X;q)
     + \widehat M_s\, g_m^{\mathrm{repo}}(X;q)
\bigr)\,dx,
\]
with overall sign fixed only through
\[
A_n = K_n - G_{\mathrm{ps},n} + B_{\mathrm{partial},n}.
\]
The current trial-side forcing terms are
\[
r(X)=c_0u_s+s_0u_n,
\qquad
r_x(X)= -\kappa_{s0}s_0u_s + c_0u_s' + \kappa_{s0}c_0u_n + s_0u_n',
\]
\[
T_\theta(X) = \nu T_s + \frac{r(X)}{x} + \frac{n}{x}v,
\qquad
e_s^{\mathrm{code}}(X) = T_s - \nu T_\theta(X),
\]
\[
g_s^{\mathrm{repo}}(X;q)
 = -\frac{T_\theta^0}{r_0}r_x(X) - T_{sn}^0\varphi',
\]
\[
g_n^{\mathrm{repo}}(X;q)
 = T_s^0\varphi' + \frac{c_0}{x}T_\theta^0\varphi
   - \frac{q^0}{x}\bigl(r_0 e_s^{\mathrm{code}}(X) + \lambda_{s0}r(X)\bigr),
\]
\[
g_m^{\mathrm{repo}}(X;q) = \frac{M_\theta^0}{r_0}r_x(X).
\]
In the active code, $T_{sn}^0$ is stored as the background quantity $Q_0$.
This closed statement is now available at repository level, but it is still not
the final article-level full formula for all of $G_{\mathrm{ps}}$.

\section{Boundary Conditions and Conjugate Pairs}
The current right-boundary conjugate pairs are
\[
(T_s,u_s), \qquad (Q_s,u_n), \qquad (S,v), \qquad (M_s,\varphi), \qquad (H,\psi).
\]
For the present mixed-weak testbench the essential conditions are
\[
u_n(1)=0, \qquad \varphi(1)=0,
\]
with free right-boundary variations in $u_s(1)$, $v(1)$, and $\psi(1)$.
The resulting natural conditions are therefore
\[
T_s(1)=0, \qquad S(1)=0, \qquad H(1)=0.
\]

This is a closed local step, but not yet a complete derivation of the whole weak
form.

\section{Current Criticality Criterion}
The current center-regular scaling statement is
\[
u_s,u_n,v = O(x^n),
\qquad
\varphi,\psi = O(x^{n-1}),
\qquad
T_s,S,T_\theta = O(x^{n-1}),
\]
\[
Q_s,M_s,M_\theta,H,\chi = O(x^{n-2}).
\]
Within the present repository boundary the physical regular family at the center
is treated as two-dimensional.
The current $v2$ workflow realizes two center-regular modes by imposing
\[
\frac{u_n}{x^n} + \frac{\lambda_c}{n}\frac{\varphi}{x^{n-1}} = 0,
\qquad
\frac{\psi}{x^{n-1}} - \lambda_c\frac{\varphi}{x^{n-1}} = 0,
\]
while normalizing one $u_s$-led mode and one $\varphi$-led mode.
The mixed-weak boundary matrix is then formed as
\[
B_{\mathrm{mix}} = B_{\mathrm{full}} V_{\mathrm{reg}},
\]
with $V_{\mathrm{reg}}$ built from those two center-regular modes rather than
from raw surrogate kernel directions.
The current working spectral criterion is
\[
\sigma_{\min}(B_{\mathrm{mix}}(q)) = 0.
\]

\paragraph{Status.}
Pilot 03 tightens the current two-dimensional center-family claim within the
reduced-center/testbench boundary.
Pilot 04 checks that $B_{\mathrm{mix}}$ is built from the two center-regular
modes.
Pilot 05 checks that $\sigma_{\min}(B_{\mathrm{mix}})$ is numerically usable as
the current working criterion in the live scan workflow.
None of these pilots proves the final closed shell BVP or the final physical
simple-support critical load.

\section{Solution Procedure}
A compact description of the present workflow is:
\begin{enumerate}[leftmargin=2em]
    \item Compute the axisymmetric prebuckling background.
    \item Build the current mixed-weak linearized operator and the reconstructed
    auxiliary channels around that background.
    \item Impose the center-regular constraints and construct the two active
    regular modes.
    \item Form $B_{\mathrm{mix}}$ from those two regular modes.
    \item Scan the load parameter and monitor $\sigma_{\min}(B_{\mathrm{mix}})$.
    \item Treat any resulting candidate load as exploratory unless the remaining
    background and full-BVP consistency issues are also resolved.
\end{enumerate}

\subsection{Still-Open Points}
The main open points remain:
\begin{itemize}[leftmargin=2em]

    \item the final precise partition between $G_{\mathrm{ps}}$ and the other

At present the leading exploratory mixed-weak candidate remains near
$n=13$ with $q\approx 3.79\ldots 3.80\,\mathrm{MPa}$, with a nearby
$n=14$ competitor near $4.28\,\mathrm{MPa}$, but these values must still be
read as exploratory rather than final.

\section{Conclusion}
The current mixed-weak branch now has a coherent repository-level structure:
active unknown/resultant blocks, verified boundary-pair logic, structurally
independent circumferential channels, a repository-level closed statement of the
prestress/load block, a two-mode center-regular construction, a boundary matrix
built from those regular modes, and a working spectral criterion based on

At the same time, the project still does not claim a final closed shell theorem,
a final closed mixed BVP, or a final physical simple-support critical load.
The purpose of this note is to keep that mixed state explicit and discussion-ready.
