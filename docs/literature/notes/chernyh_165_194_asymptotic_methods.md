# Черных и др., стр. 165–194 — асимптотические методы в нелинейной теории оболочек

**Источник:** `Chernyh, Kabric, Mihajlovskij, Tovstik, Shamin. Obshchaja n-165-194.pdf`

## 1. Что именно важно для проекта
- Источник важен для понимания того, как asymptotic reasoning легально используется в nonlinear shell theory.
- Для твоего проекта это не первый файл для запуска solver’а, но хороший источник для:
  - локальных/asymptotic checks;
  - понимания scaling near edge / center;
  - методического обоснования различных limit regimes.

## 2. Какие обозначения используются
- Вероятно, asymptotic notation книги Черныха для shell expansions и limit transitions.
- Это не рабочая solver-нотация, а методический слой.

## 3. Какие уравнения / страницы критичны
- **Весь диапазон 165–194**.
- Ищи:
  - общие asymptotic schemes;
  - boundary-layer type arguments;
  - scale separation between membrane/bending effects.
- Особенно полезно читать после 60–76 и 150–164.

## 4. Где возможны расхождения с твоей моделью
- Этот файл сам по себе не задаст тебе конечную mixed-weak систему.
- Ошибка будет, если заменить им полноценный вывод уравнений.
- Правильное использование: как method note and asymptotic support.

## 5. Можно ли использовать как основной источник
- **Нет, не как единственный основной.**
- **Да, как важный вспомогательный методический источник.**
- Статус: **вспомогательный среднего/высокого приоритета**.
