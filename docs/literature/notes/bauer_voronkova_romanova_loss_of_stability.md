# Bauer–Voronkova–Romanova — О потере устойчивости симметричных форм равновесия круглых пластин

**Источник:** `BauerVoronkovaRomanova.pdf`

## 1. Что именно важно для проекта
- Полезен как plate-level источник, где явно прослеживается схема:
  - axisymmetric base state,
  - линеаризация для неосесимметрических мод,
  - поиск критической нагрузки и номера волны.
- Особенно полезен для понимания того, как radial inhomogeneity влияет на `p_cr` и `n_cr`.
- Для проекта это хороший reference, если нужно сравнить, как в классической plate-задаче с неоднородностью возникает circumferential compression near edge.

## 2. Какие обозначения используются
- Переменные `ω_0`, `φ_0` для осесимметрического состояния и `w_n`, `F_n` для неосесимметрической части.
- Присутствуют модифицированные операторы `L_1`, `L_2`, `Δ_n`.
- Нотация типична для nonlinear plate theory, а не для shell mixed-weak formulation.

## 3. Какие уравнения / страницы критичны
- **С. 2** — operators `L_1`, `L_2` и базовые BC.
- **С. 3** — осесимметрическая система `(6)–(7)` и линеаризованная система для `w_n`, `F_n` в формуле `(8)`.
- **С. 4** — circumferential compression near edge and interpretation.
- **С. 4–5** — таблицы/графики для `p_cr` и `n`.
- **С. 6** — выводы: weakening toward edge lowers critical load and promotes unsymmetric transition.

## 4. Где возможны расхождения с твоей моделью
- Это plate model, не общая теория оболочек.
- Нет твой mixed-weak block, нет `H`, `S`, `chi`, нет boundary matrix approach.
- Использовать как прямой источник для solver’а shell-type simple support нельзя.

## 5. Можно ли использовать как основной источник
- **Нет, как основной выводной источник — нет.**
- **Да, как вспомогательный источник для plate-level comparison и для qualitative trends.**
- Статус: **вспомогательный**.
