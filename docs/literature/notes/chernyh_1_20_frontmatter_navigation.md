# Черных и др., стр. 1–20 — титул / оглавление / вводные страницы

**Источник:** `Chernyh, Kabric, Mihajlovskij, Tovstik, Shamin. Obshchaja n-1-20.pdf`

## 1. Что именно важно для проекта
- Этот файл полезен в основном как **навигационный**: титул, оглавление, расположение глав, структура книги.
- Для Codex это важно, потому что позволяет быстро понять, **в какой именно главе искать** геометрию, осесимметрическую деформацию и asymptotic methods.

## 2. Какие обозначения используются
- На этих страницах нет смысла фиксировать рабочие обозначения проекта.
- Использовать их как источник формул не надо.

## 3. Какие уравнения / страницы критичны
- Критичны не формулы, а **оглавление** и названия глав.
- Именно отсюда удобно понять, что:
  - стр. 21–40 — базовая нелинейная теория упругости,
  - стр. 60–76 — осесимметричная деформация оболочки вращения,
  - стр. 150–164 — нелинейная теория первого приближения,
  - стр. 165–194 — асимптотические методы.

## 4. Где возможны расхождения с твоей моделью
- Это не формульный источник, поэтому сравнивать с моделью здесь почти нечего.
- Ошибка будет, если начать ссылаться на этот файл как на источник конкретных equations.

## 5. Можно ли использовать как основной источник
- **Нет.**
- Статус: **навигационный / служебный**.
