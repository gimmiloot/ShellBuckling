# Bauer–Voronkova–Semenov 2022 — К вопросу о несимметричных формах равновесия круглых пластин

**Источник:** `BauerVoronkovaSemenov-vestnik2022_1.pdf`

## 1. Что именно важно для проекта
- Это один из самых важных для твоего текущего проекта источников **по физической интерпретации проблемы simple support / hinged support**.
- Статья одновременно рассматривает analytical model и FEM/ANSYS, что делает её очень полезной для sanity-check твоих результатов.
- Главная ценность для проекта: авторы прямо показывают, что при **шарнирном опирании** и больших нагрузках shallow-shell / plate-level докритическое состояние может описываться плохо, и без более общей теории оболочек результаты расходятся.

## 2. Какие обозначения используются
- Базовая постановка идёт через classical plate variables `w(r,θ)` и stress function `F(r,θ)`.
- После выделения осесимметрической части возникают `w_0`, `F_0` (или близкие обозначения), а для неосесимметрической части — `w_n(r)`, `F_n(r)`.
- Граничные условия отдельно выписываются для **скользящей заделки** и **шарнирного опирания**.

## 3. Какие уравнения / страницы критичны
- **С. 2** — базовая система уравнений для пластины под нормальным давлением.
- **С. 3** — граничные условия для шарнирного опирания и линеаризованная система для `w_n`, `F_n`.
- **С. 4–5** — FEM постановка в ANSYS, описание imperfections и способа определения потери устойчивости.
- **С. 5–7** — главное обсуждение различий между sliding clamp и hinged support.
- Особенно важен вывод, что при шарнирном опирании критические явления и субкритическое состояние требуют **более точного описания**, чем даёт shallow-shell approximation.

## 4. Где возможны расхождения с твоей моделью
- Это still plate / shallow-shell language, а не твой mixed-weak shell class.
- В статье нет твоих независимых окружных mixed variables и `B_mix`.
- Но физический вывод статьи как раз поддерживает твою текущую рабочую гипотезу: проблема в полной simple-support задаче, скорее всего, сидит в **background solver’е / точности субкритического состояния**, а не просто в одной правой граничной строке.

## 5. Можно ли использовать как основной источник
- **Да, как основной физико-интерпретационный источник по hinged/simple-support линии проекта.**
- **Нет, как единственный выводной источник формул mixed-weak модели.**
- Статус: **основной для интерпретации и сверки с FEM**, **вспомогательный для вывода формул**.
