# Григолюк — Большие осесимметричные прогибы оболочек вращения

**Источник:** `Grigoluk-107-147.pdf`

## 1. Что именно важно для проекта
- Это один из наиболее естественных источников для твоей линии **осесимметрического фона**.
- По названию и месту в книге/сборнике это именно тот тип материала, который полезен для:
  - нелинейного осесимметрического BVP;
  - continuation по нагрузке;
  - понимания больших прогибов оболочек вращения до bifurcation layer.
- Если сейчас главная практическая проблема сидит в `simple-support` background solver’е, то этот источник должен считаться одним из приоритетных.

## 2. Какие обозначения используются
- В имеющемся OCR текст почти не читается, поэтому обозначения надо брать **напрямую из PDF при сверке**.
- По теме источника почти наверняка используются классические для оболочек вращения переменные: меридиональная координата, радиус, прогиб/перемещения, усилия и моменты.
- Для Codex-заметок важно зафиксировать: этот файл нужен **не для current mixed-weak symbols**, а для осесимметрического baseline.

## 3. Какие уравнения / страницы критичны
- **Весь диапазон 107–147** следует считать потенциально важным.
- В первую очередь искать:
  - постановку осесимметрической деформации оболочки вращения;
  - краевые условия;
  - переход к большим прогибам;
  - условия потери устойчивости / предельной точки.
- При следующем проходе по этому PDF стоит отдельно выписать точные номера формул после ручной проверки скана.

## 4. Где возможны расхождения с твоей моделью
- Источник, вероятно, не содержит твоей текущей mixed-weak boundary-matrix formulation.
- Но он может быть ближе к нужной физике осесимметрического background, чем plate-level статьи.
- Главный риск — не перепутать “источник для осесимметрического фона” с “источником для неосесимметрической mixed-weak части”.

## 5. Можно ли использовать как основной источник
- **Да, как основной источник для осесимметрического background слоя.**
- Нужна повторная ручная проверка точных формул по скану.
- Статус: **основной, но требует ручной верификации по PDF**.
